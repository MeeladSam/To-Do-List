# My Super Simple To-Do List

This is a very basic To-Do List app made with Java Swing.  
It’s nothing fancy, just a small project for learning and messing around.

### What it does:
- You can type a task in the text box and click Add to put it in the list.
- Select a task and click Done to mark it as done. The task disappears after 2 seconds.
- Select a task and click Edit to change its name.
- Each task has a light line below it so the list looks a bit organized.

### How to use:
1. Open the project in any Java IDE like Eclipse or IntelliJ.
2. Type a task and click Add.
3. Select a task and click Done or Edit to play with it.
